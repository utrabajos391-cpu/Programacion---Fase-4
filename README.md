# Sistema Integral de Gestión de Clientes, Servicios y Reservas - Software FJ

## Descripción

Este proyecto corresponde a la Fase 4 del curso Programación 213023 de la Universidad Nacional Abierta y a Distancia UNAD.

El sistema fue desarrollado en Python aplicando programación orientada a objetos y manejo avanzado de excepciones. La aplicación permite gestionar clientes, servicios y reservas para la empresa Software FJ, sin utilizar bases de datos. La información se administra mediante objetos, listas internas y un archivo de logs para registrar eventos y errores.

## Funcionalidades principales

- Registro de clientes con validaciones.
- Creación de servicios especializados.
- Reserva de salas.
- Alquiler de equipos.
- Asesorías especializadas.
- Confirmación, cancelación y procesamiento de reservas.
- Manejo de excepciones personalizadas.
- Registro de errores y eventos en archivo de logs.
- Simulación de más de 10 operaciones válidas e inválidas.

## Principios aplicados

- Abstracción.
- Herencia.
- Polimorfismo.
- Encapsulación.
- Manejo de listas internas.
- Excepciones personalizadas.
- Bloques try, except, else y finally.
- Registro de eventos en archivo externo.

## Archivos del proyecto

- `Sistema_Gestión.py`: contiene el código fuente principal.
- `logs_sistema.txt`: archivo generado automáticamente con eventos y errores.
- `README.md`: descripción general del proyecto.
